Cadeia de Agentes Especializados

Cada agente faz UMA coisa muito bem. O output de um vira input do proximo. Resultado: sinergia total.

1

Brand Strategist

Decisor

O estrategista. Analisa o briefing e toma as decisoes de alto nivel.

RECEBE:
Brief do usuario, config da marca, objetivo
PRODUZ:
Template escolhido, arco emocional, constraints para todos os outros agentes
2

Story Architect

Estruturador

O arquiteto da historia. Define o proposito de cada slide e como eles se conectam.

RECEBE:
Blueprint do Strategist + brief original
PRODUZ:
Estrutura slide-by-slide com proposito, beat emocional, direcao visual
3

Copywriter

Escritor

O redator. Escreve headlines, body copy e CTAs respeitando limites de caracteres e tom da marca.

RECEBE:
Estrutura do Architect + constraints de copy (max chars, estilo)
PRODUZ:
Copy completo para cada slide: headline (max 60 chars), body, bullets, CTA
4

Visual Compositor

Designer

O diretor de arte. Especifica cores, fontes, posicoes - tudo pixel-perfect.

RECEBE:
Copy do Copywriter + paleta da marca + layouts disponiveis
PRODUZ:
Specs visuais: layout ID, cores, fontes, posicao X/Y de cada elemento
5

Quality Validator

Validador

O controle de qualidade. Valida se tudo esta on-brand: cores, fonts, contraste, voz.

VALIDA:
Headlines presentes? Cores no palette? Contraste WCAG AA? Tom alinhado?
PRODUZ:
Score 0-100, lista de checks, warnings, erros criticos se houver
6

Render Engine

Finalizador

O renderizador. Transforma specs em HTML/CSS real, pronto pra exportar.

RECEBE:
Visual specs do Compositor (layout ID, posicoes, estilos)
PRODUZ:
HTML + CSS completo, preview de cada slide, pronto pra export PNG
Configuracao de Marca

3 Pilares do Brand Config

Configure uma vez, use sempre. O sistema CONHECE sua marca.

Identidade Visual

40% do score de completude

  • Logo (PNG/SVG)
  • Cor primaria, secundaria, accent
  • Fonte heading + body
  • WCAG compliance

Brand Voice

30% do score de completude

  • Atributos (professional, bold, etc)
  • Tone guidelines
  • Exemplos de copy (bom/ruim)
  • Frases a evitar

Exemplos

30% do score de completude

  • Upload de carousels existentes
  • Ads, posts, slides
  • Anotacoes do que e on-brand
  • Minimo 3 exemplos
